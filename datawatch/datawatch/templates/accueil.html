Bienvenue sur DataWatch

Analyse Automatique de Fichiers CSV

DataWatch est un projet de pipeline d’analyse de données entièrement serverless, conçu pour analyser automatiquement des fichiers CSV contenant des données cruciales pour la prise de décision dans les entreprises.

Le système détecte des anomalies dans les données, génère des rapports détaillés et envoie des notifications de manière automatisée.

Fonctionnalités

- Dépôt facile des fichiers CSV via une interface web.
- Analyse automatique des fichiers sur un service cloud serverless (AWS, Azure, GCP).
- Détection d’anomalies dans les données et génération de rapports.
- Visualisation des résultats via une interface web.
- Notification des récapitulatifs par email ou push.

Objectifs Pédagogiques

- Mettre en œuvre un workflow d’analyse serverless dans un cloud public.
- Exploiter des services de stockage cloud pour gérer les fichiers entrants et sortants.
- Déployer des fonctions serverless (FaaS) pour traiter les données sans infrastructure.
- Automatiser l’ensemble du traitement des fichiers sans intervention humaine.
